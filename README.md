# Furby 2012 / Boom Audio & Image Extraction

This repository contains exported audio and image assets from multiple Furby variants, along with tooling for extracting and replacing those assets within firmware binaries.

## 📁 Included Files

All extracted assets can be found in: Furby Exported Files/

Supported Furby variants:

- Dutch 2012  
- English 2012  
- French 2012  
- English Boom  
- French Boom  
- English Crystal *(same as Boom)*  
- Furbacca  

These folders contain raw audio clips and image assets extracted directly from firmware.

## ⚙️ Tools & Code

The remaining code in this repository is used for:

- Extracting audio and image data from firmware binaries  
- Replacing modified assets back into the binary  

Documentation for these tools is planned but not yet available.
